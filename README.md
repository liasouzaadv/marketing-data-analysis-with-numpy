# Análise Estatística de Dados com NumPy aplicada à área de Marketing

##  Contexto do Projeto
Este projeto foi desenvolvido como parte do curso **Fundamentos da Linguagem Python – Do Básico a Aplicações de IA**, oferecido pela **Data Science Academy**.

O objetivo do mini-projeto é aplicar **conceitos fundamentais de estatística e computação numérica em Python**, utilizando **NumPy**, para analisar dados simulados relacionados ao comportamento de usuários em um contexto de marketing digital.

---

##  Objetivo
Simular e analisar o comportamento de usuários em um ambiente de marketing, respondendo perguntas como:
- Qual o tempo médio que os usuários passam no site?
- Existe relação entre número de visitas e valor de compra?
- Usuários mais engajados tendem a gastar mais?
- Quais padrões podem ser observados a partir de dados estatísticos?

---

##  Dados Utilizados
Os dados são **sintéticos**, gerados artificialmente com NumPy, simulando:
- Tempo de navegação no site
- Número de visitas
- Quantidade de itens no carrinho
- Valor final da compra

Essa abordagem permite focar na **análise estatística**, sem dependência de bases externas.

---

##  Por que NumPy em vez de Pandas?
Embora bibliotecas como Pandas sejam amplamente utilizadas em análises de dados, este projeto foi desenvolvido exclusivamente com **NumPy** para reforçar os **fundamentos da computação numérica e estatística**, conforme a proposta do curso.

O projeto envolve:
- geração de dados sintéticos
- aplicação de distribuições estatísticas
- operações matemáticas vetorizadas
- análise de correlação entre variáveis numéricas

Nesse contexto, o NumPy oferece:
- maior **controle sobre os dados**
- melhor **performance em cálculos matemáticos**
- compreensão aprofundada de como os dados são processados internamente

Além disso, o NumPy é a **base do ecossistema de ciência de dados em Python**, sendo fundamental para bibliotecas como Pandas, SciPy e Scikit-learn.

##  Análises Realizadas
- Estatística descritiva (média, mediana, desvio padrão)
- Simulação de comportamento de usuários
- Análise de correlação entre variáveis
- Segmentação de usuários com base em critérios numéricos
- Visualização de dados para apoio à interpretação dos resultados
